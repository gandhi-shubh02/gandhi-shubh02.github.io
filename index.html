Language-to-SQL Generation using T5 Fine-Tuning and LLM Prompting

May 2025

GitHub Repository

Built a natural language to SQL translation system on the ATIS flight booking dataset using three approaches: fine-tuning a pre-trained T5 encoder-decoder, training the same architecture from scratch, and designing zero- and few-shot prompts for in-context learning with instruction-tuned Gemma LLMs. Explored the impact of tokenizer design, parameter freezing, and data preprocessing on performance. Conducted ablation analysis on prompt design and evaluated models using Record-level F1, SQL Exact Match, and execution-based metrics.

Image Generation and Editing with Diffusion Models (DeepFloyd, DDPM)

April 2025

GitHub Repository

Implemented and deployed image generation pipelines using pre-trained DeepFloyd IF and a custom-trained DDPM. Developed forward and reverse sampling loops to support denoising, inpainting, text-guided editing,image-to-image translation, and visual illusions. Trained a time- and class-conditioned UNet on MNIST, achieving high visual fidelity across noise levels and out-of-distribution samples. Enhanced generation quality through classifier-free guidance and optimized denoising with strided schedules.
